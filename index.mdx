Dapr is a portable, event-driven runtime that makes it easy for any developer to build resilient, stateless and stateful applications that run in cloud and edge environments. It provides a set of building blocks as HTTP and gRPC APIs, deployed as a sidecar alongside your application — no special SDKs or libraries required.

## Why Dapr?

Dapr solves the hard problems of distributed systems so you don't have to write boilerplate infrastructure code. Every building block is exposed as a standard HTTP or gRPC API — your application calls `localhost` and Dapr handles the rest.
